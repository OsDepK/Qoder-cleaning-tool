## 🚀 快速开始

### 第一步：使用Pycharm安装依赖

```bash
pip install PyQt5
```

### 第二步：运行程序

**Windows:**
```cmd
python qoder_reset_gui.py
```
或双击运行 `start_gui.bat`

**macOS/Linux:**
```bash
python3 qoder_reset_gui.py
```
或运行 `./start_gui.sh`

### 第三步：选择功能

推荐使用 **"一键重置配置"** 按钮，这是最全面的清理方案。

## ⚙️ 功能说明

### 🔵 一键重置配置（推荐）

**最全面的清理方案，包含以下所有功能：**

- ✅ 重置设备ID
- ✅ 重置遥测数据
- ✅ 清理缓存数据
- ✅ 清理身份识别文件
- ✅ 执行高级身份清理
- ✅ 清理登录信息
- ✅ 硬件指纹重置
- ✅ 智能对话管理

**使用场景**: 首次使用或需要彻底重置时

---

### 🔴 关闭Qoder

检查 Qoder 应用程序的运行状态。

**使用场景**: 执行任何清理操作前，确保 Qoder 已完全关闭

---

### 💻 重置设备ID

生成全新的设备标识符，让系统识别为新设备。

**清理内容**:
- `machineid` 文件
- `deviceid`, `hardware_uuid`, `system_uuid` 等文件
- `storage.json` 中的机器ID

**使用场景**: 只需重置设备标识时

---

### 📊 重置遥测数据

重置应用程序的遥测信息，包括设备ID、会话ID等。

**清理内容**:
- `telemetry.machineId`
- `telemetry.devDeviceId`
- `telemetry.sessionId`
- `telemetry.installationId`
- 其他遥测标识符

**使用场景**: 只需重置遥测数据时

---

### 🔥 深度清理身份

清理所有网络状态、Cookie、本地存储等身份识别信息。

**清理内容**:
- 网络持久状态
- 信任令牌
- 共享存储
- 本地存储数据
- SharedClientCache 内部文件

**使用场景**: 需要彻底清理身份信息时

**注意**: 此功能会清除对话记录

---

### 🔐 清理登录信息

专门清理登录相关的身份信息，包括认证令牌、会话数据等。

**清理内容**:
- 登录证书和 Cookies
- SharedClientCache 登录状态
- 网络状态和会话存储
- 设备认证信息
- nonce 和 challenge 数据

**使用场景**: 只需清理登录状态时

---

### 🛡️ 硬件指纹重置

生成虚假硬件信息，干扰设备指纹检测。

**功能特点**:
- 重置硬件标识符（CPU、GPU、内存等）
- 根据系统类型生成相应硬件配置
- 创建多个虚假硬件信息文件
- 重置系统版本、架构等指纹

**使用场景**: 需要最强反检测效果时

**重要提示**: 建议执行后重启系统

---

## 📝 使用流程

### 标准使用流程

1. **关闭 Qoder**
   - 完全退出 Qoder 应用程序
   - 可使用 "关闭Qoder" 按钮检查状态

2. **启动 Clean-Qoder**
   - 运行程序后会自动进行状态检查
   - 查看日志了解当前系统状态

3. **选择清理选项**
   - 决定是否保留聊天记录（默认保留）
   - 选择具体的清理功能

4. **执行清理**
   - 点击相应按钮
   - 确认操作对话框
   - 等待操作完成

5. **重新启动 Qoder**
   - 清理完成后重新打开 Qoder
   - 使用新身份注册或登录

### 推荐方案

**最彻底的清理方案：**
1. 取消勾选 "保留聊天记录"
2. 点击 "一键重置配置"
3. 确认执行所有清理操作
4. 重启系统（可选，但推荐）
5. 重新启动 Qoder

**快速清理方案：**
1. 保持勾选 "保留聊天记录"
2. 点击 "一键重置配置"
3. 重新启动 Qoder

## ❓ 常见问题

### Q1: 提示 "未安装 PyQt5"？

**解决方案：**
```bash
pip install PyQt5
# 或
pip3 install PyQt5
```

---

### Q2: 提示 "Qoder 正在运行"？

**解决方案：**
1. 完全关闭 Qoder 应用程序
2. Windows: 使用任务管理器确认进程已结束
3. macOS: 使用 Cmd+Q 退出，或从菜单选择退出
4. 再次点击 "关闭Qoder" 按钮确认

---

### Q3: 重置后仍被识别为老用户？

**可能原因和解决方案：**

1. **未执行完整清理**
   - 使用 "一键重置配置" 而不是单个功能
   - 确保所有清理步骤都已完成

2. **未清理关键文件**
   - 取消勾选 "保留聊天记录"
   - 使用 "深度清理身份" 功能
   - 使用 "硬件指纹重置" 功能

3. **网络环境相同**
   - 考虑更换网络环境
   - 使用 VPN 或其他网络

4. **系统未重启**
   - 执行 "硬件指纹重置" 后建议重启系统

5. **浏览器缓存**
   - 重置后使用指纹浏览器注册
   - 避免使用有缓存的浏览器

---

### Q4: 找不到 Qoder 目录？

**检查步骤：**
1. 确认已安装 Qoder 应用程序
2. 至少运行过一次 Qoder（创建配置目录）
3. Windows: 检查 `%APPDATA%\Qoder` 目录
4. macOS: 检查 `~/Library/Application Support/Qoder` 目录

---

### Q5: 操作日志显示错误？

**排查方法：**
1. 检查是否已完全关闭 Qoder
2. 确认对 Qoder 目录有读写权限
3. 查看详细错误信息
4. 尝试以管理员权限运行（如需要）

---

### Q6: 如何切换语言？

点击界面右上角的语言下拉菜单，选择：
- 中文
- English
- Русский
- Português (BR)
- 日本語
- 한국어

界面和提示信息会立即更新为所选语言。

## ⚠️ 注意事项

### 使用前必读

1. **备份重要数据**
   - 建议在执行清理前备份重要的 Qoder 配置
   - 对话记录（如未勾选保留）将被永久删除

2. **关闭应用程序**
   - 执行任何清理操作前，必须完全关闭 Qoder
   - 使用 "关闭Qoder" 按钮确认状态

3. **重新登录**
   - 清理完成后需要重新登录 Qoder 账户
   - 可能需要重新输入账号密码

4. **合法使用**
   - 本工具仅供学习和研究使用
   - 请遵守相关法律法规和软件使用协议

5. **系统重启**
   - 执行 "硬件指纹重置" 后建议重启系统
   - 重启后效果更佳

6. **注册建议**
   - 重置后注册建议使用指纹浏览器
   - 避免使用有缓存的普通浏览器
